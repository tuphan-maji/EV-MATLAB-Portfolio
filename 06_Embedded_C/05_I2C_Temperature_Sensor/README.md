# I2C Temperature Sensor Interface Using Embedded C

## Objective

The objective of this project is to develop an Embedded C program that interfaces with a digital temperature sensor using the I2C (Inter-Integrated Circuit) protocol. The program reads temperature data from the sensor, converts the raw value into degrees Celsius, and displays the measured temperature. This project demonstrates the basic principle used in embedded systems, Battery Management Systems (BMS), Vehicle Control Units (VCU), and automotive Electronic Control Units (ECUs) for temperature monitoring and thermal protection.

## Theory

I2C is a synchronous two-wire serial communication protocol widely used for communication between a microcontroller and peripheral devices such as temperature sensors, EEPROMs, RTCs, ADCs, and displays.

I2C communication uses two signals:

* SDA (Serial Data Line) – Bidirectional data line.

* SCL (Serial Clock Line) – Clock signal generated by the master device.

Each I2C device has a unique address. The microcontroller initiates communication by sending a START condition followed by the sensor address and register address. The sensor then returns the temperature data.

In this project, the communication sequence is:

* Send START condition.

* Send sensor I2C address.

* Send temperature register address.

* Read two bytes of temperature data.

* Combine the bytes into a 16-bit value.

* Convert the raw value into degrees Celsius.

* Display the temperature.

This approach is widely used in EV battery thermal monitoring because it provides reliable and low-cost temperature sensing.

## Mathematical Representation

### Raw Temperature Combination

Temp_raw = (MSB << 8) | LSB

### Temperature Conversion

Temperature (°C) = Temp_raw × 0.0625

## Input Parameters

| Parameter            | Value         |
| -------------------- | ------------- |
| Sensor I2C Address   | 0x48          |
| Temperature Register | 0x00          |
| MSB                  | 0x19          |
| LSB                  | 0x00          |
| Resolution           | 0.0625 °C/bit |

## Program Flow

* Start

* Initialize I2C peripheral.

* Send START condition.

* Send sensor address (0x48).

* Send temperature register address (0x00).

* Generate repeated START.

* Read MSB byte.

* Read LSB byte.

* Combine bytes into 16-bit value.

* Convert raw data to Celsius.

* Display temperature.

* Stop I2C communication.

* End

## Calculation

| MSB           | 0x19                        |
| ------------- | --------------------------- |
| LSB           | 0x00                        |
| Raw Value     | (0x19 << 8) | 0x00 = 0x1900 |
| Shifted Value | 0x1900 >> 4 = 0x190 = 400   |
| Temperature   | 400 × 0.0625 = 25.0 °C      |
