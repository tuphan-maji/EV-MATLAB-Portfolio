# Vehicle Tractive Force Analysis

## Objective
The objective of this project is to calculate the total tractive force required by an electric vehicle under different driving conditions and to plot the tractive force versus vehicle speed using MATLAB.

## Theory

The total tractive force required to move an electric vehicle is the sum of four forces:

1. Rolling Resistance Force
2. Aerodynamic Drag Force
3. Gradient Force
4. Acceleration Force

Total Tractive Force:

Ft = Frr + Fad + Fg + Fa

## Input Parameters

| Parameter | Value |
|-----------|-------|
| Vehicle Mass | 1500 kg |
| Gravity | 9.81 m/s² |
| Rolling Resistance Coefficient | 0.015 |
| Drag Coefficient | 0.30 |
| Frontal Area | 2.2 m² |
| Air Density | 1.225 kg/m³ |
| Road Gradient | 5° |
| Acceleration | 2 m/s² |
| Speed Range | 0–40 m/s |

## MATLAB Files

Vehicle_Tractive_Force.m

## Output

The MATLAB program calculates:

- Rolling Resistance Force
- Aerodynamic Drag Force
- Gradient Force
- Acceleration Force
- Total Tractive Force

It also plots:

Tractive Force vs Vehicle Speed
